## 使用方法

```
$ git clone https://github.com/GuoYongfeng/github-notetaker-app
$ cd github-notetaker-app
$ npm install
$ npm run dev
```

产出生产环境资源
```
$ npm run build
```

## 技术点

- webpack && webpack-dev-server
- babel 及其各种插件
- ES6语法特性
- React && ReactDOM
- react-router
- axois
- firebase && reactfire
